# ComicReader
### Require:
- .Net Core 3.1

### Features:
- Tree view to choose path
- Chapter next/previous button
- Zoom up/down images of chapter
- View path to see current choosing path
### Keys:
- j: Scroll down
- k: Scroll up
- i: Zoom out
- o: Zoom in
